# Run-Tracker
This is a Java project that uses the Spring Boot library and frame work to track runs. Runs are stored in an database.

This project uses:
- Java 21
- Spring Boot 3.3.5
- Maven
- Dependencies : Spring Boot Dev Tools
- Dependencies : Spring Web
- Packaging : Jar
